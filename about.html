A university cloud computing lab bringing real-world cloud skills to students through hands-on practice and live deployments.

Our Mission

Bridging Theory and Real Cloud Practice

This project is designed to help students understand the fundamentals of cloud computing — from the basics of virtualization to deploying live applications on modern platforms like GitHub Pages.

Students gain practical experience working with real cloud tools, exploring how websites are hosted, managed, and scaled in production environments.

Meet Our Team →

What You'll Learn

  • Fundamentals of cloud infrastructure and service models (IaaS, PaaS, SaaS)
  • How to deploy static and dynamic web apps to the cloud
  • Web hosting, CDN distribution, and DNS management
  • Virtualization concepts and containerization basics
  • Real-world cloud platform usage (GitHub, AWS concepts)
  • Security best practices for cloud-hosted applications
Core Topics

Lab Curriculum

☁

Cloud Basics

Understand IaaS, PaaS, SaaS, and the architecture of modern cloud data centers.

💻

Practical Labs

Hands-on deployment experiments pushing apps live to real cloud environments.

🚀

Modern Hosting

Learn GitHub Pages, static hosting, CDN delivery, and custom domain setup.

Containerization

Introduction to Docker and how containers revolutionize cloud deployments.

🔒

Cloud Security

HTTPS, IAM roles, access policies, and securing cloud-deployed applications.

📡

Networking

Virtual networks, subnets, load balancers, and cloud-based DNS management.
